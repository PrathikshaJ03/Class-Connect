Class Connect

Student-Teacher Portal

About Us

A Class Connect is a web-based platform that provides a communication and collaboration tool for students and teachers. It can serve as a centralized hub for managing assignments, grades and other academic information. The goal of a student-teacher portal is to improve communication, enhance collaboration, and increase transparency between students and teachers. Features may include online submissions of assignments, access to grades and communication tools such as messaging and information sharing about events and activities.

Our Services

For Teachers

  1. Teachers can add students in their class through Add Student option
  2. Teachers can give or edit their class students marks
  3. Teachers can write notice which will be send to all the students in their class
  4. Teachers can upload assignments through upload assignment option
  5. Teachers can check their student's assignment submission and give them marks accordingly
  6. Students can message Teachers directly and their messages can be seen in Inbox
  7. User can add profile picture through edit profile option

For Students

  1. Students can login to the portal and keep themselves updated of all the activities going on in the class
  2. Students can see the notice given by their teacher in the Class Notice option
  3. Students can download their assignments in the Assignments option and can also submit it
  4. Students can contact any teacher through message option in All Teachers List
  5. Students can check their marks in the Marks option
  6. User can add profile picture through edit profile option
